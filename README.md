### Prerequisites

In order to run this project you need:

- Visual Studio
- Ruby on Rails
- React & Redux
- Postgresql

### Setup

Clone this repository to your desired folder:

```sh
  git@github.com:gak92/hello-rails-react.git
```

### Setup Database

```sh
  rails db:create
  rails db:migrate
  rails db:seed
```

### Install

Install this project with:

```sh
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  yarn build
  rails server
```
